# Sprint 3: UI Foundation & Styling

## Goal

Integrate NativeWind (Tailwind CSS) and establish a "shadcn-like" component library.

## User Stories

- **As a Developer**, I want to use Tailwind CSS utility classes so that I can style components quickly and consistently.
- **As a Developer**, I want a set of reusable UI components (Button, Card, Typography) to ensure a polished look.

## Tech Stack

- **NativeWind**: Tailwind CSS for React Native.
- **React Native Reusables**: shadcn-like component library for RN.

## Tasks

- [ ] Install and configure `nativewind` and `tailwindcss`.
- [ ] Install and configure `@react-native-reusables` components.
- [ ] Create `src/components/ui/` folder.
- [ ] Implement base components: `Button`, `Text`, `Heading`, `Card` (or use reusables).
- [ ] Ensure components support both Light and Dark modes.

## Definition of Done

- Tailwind classes work in `.tsx` files.
- A "Components" demo page shows all implemented UI elements.
