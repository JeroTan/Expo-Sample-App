# Sprint 1: Hello World & Initial Setup

## Goal

Establish a working Expo environment and verify the app runs on a physical device or emulator.

## User Stories

- **As a Developer**, I want to initialize the Expo project so that I have a clean foundation for the mobile app.
- **As a Developer**, I want to see a "Hello World" screen on my phone so that I know the development environment is correctly configured.

## Tasks

- [x] Initialize Expo project with TypeScript template.
- [ ] Verify `npm start` works and generates a QR code.
- [ ] Connect physical device via Expo Go.
- [ ] Display a simple "Monster Hunter Weakness Viewer" greeting on the main screen.

## Definition of Done

- The app displays "Hello World" or a welcome message on a mobile device.
- `docs/AGENT_MEMORY.md` is updated with setup details.
