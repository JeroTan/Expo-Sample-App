# Sprint 6: Monster List & Weakness Details

## Goal

Implement the core value of the app: viewing monster weaknesses.

## User Stories

- **As a User**, I want to see a list of monsters for a specific game so I can find my target.
- **As a User**, I want to click a monster to see its elemental and status weaknesses.

## Tasks

- [ ] Build the Monster List screen (Searchable list with monster icons).
- [ ] Create the Monster Detail screen.
- [ ] Implement the Weakness View (Grid/Table showing Fire, Water, Ice, etc. ratings).
- [ ] Add Hitzone data if available in the API.

## Definition of Done

- User can navigate from List -> Detail.
- Weakness information is clearly displayed and accurate.
