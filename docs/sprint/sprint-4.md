# Sprint 4: Navigation & Main Menu

## Goal

Implement the main navigation structure including game selection and a settings menu.

## User Stories

- **As a User**, I want to choose between different games (World, Rise, Wilds) on the main menu.
- **As a User**, I want to access a burger/drawer menu to change settings like Dark/Light mode.

## Tasks

- [ ] Configure `expo-router` for Drawer navigation.
- [ ] Create the Main Menu screen with game selection cards.
- [ ] Implement a Drawer menu with "Appearance" toggle.
- [ ] Add the "Burger" button to the header.

## Definition of Done

- User can switch between Dark and Light mode via the menu.
- The main menu displays options for World, Rise, and Wilds.
- Navigation flows correctly between screens.
